# インストールガイド

## 配布形式

applanch は 2 種類の形式で配布されています。

- ポータブル ZIP パッケージ
- インストーラー EXE パッケージ

## ポータブル ZIP

1. GitHub Releases から ZIP アセットをダウンロードします。
2. 任意のフォルダーに展開します。
3. `applanch.exe` を実行します。

## インストーラー EXE

1. GitHub Releases からインストーラーアセットをダウンロードします。
2. インストーラーを起動します。
3. 現在のユーザーのみ、またはすべてのユーザー向けにインストールするかを選択します。
4. セットアップウィザードの手順に従います。
