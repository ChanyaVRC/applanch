# Getting Started

## What is applanch?

applanch is a lightweight Windows launcher for quickly starting apps, files, and folders.
It focuses on fast launch workflows with category-based organization, quick-add suggestions, and built-in update checks.

## Requirements

- Windows 10 or Windows 11

## Install

See the [Install Guide](../install.md) for full instructions.
After installing, run **applanch.exe** to open the main window.

## Add Your First Item

1. Type a file path, folder path, or application name into the quick-add input at the top of the window.
2. applanch shows up to 50 suggestions below the input as you type by default.
3. Select a suggestion or press **Enter** / click **Add** to add the item to your list.

You can also drag a file or folder from File Explorer and drop it onto the applanch window to add it instantly.

## Launch an Item

Click any item in the list to launch it.
