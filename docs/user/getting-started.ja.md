# クイックスタート

## applanch とは？

applanch は、アプリ・ファイル・フォルダーを素早く起動するための軽量 Windows ランチャーです。
カテゴリーによる整理、クイック追加サジェスト、組み込みのアップデート確認機能を備えたデスクトップアプリです。

## 動作要件

- Windows 10 または Windows 11
- .NET 10 ランタイム（リリース版の配布物は自己完結型のため、別途インストール不要）

## インストール

詳細な手順は [インストールガイド](../install.ja.md) を参照してください。
インストール後は **applanch.exe** を実行してメインウィンドウを開きます。

## 最初のアイテムを追加する

1. ウィンドウ上部のクイック追加入力欄に、ファイルパス・フォルダーパス・アプリ名を入力します。
2. 入力に応じて最大 10 件のサジェストが表示されます。
3. サジェストを選択するか、**Enter** を押すか、**追加** ボタンをクリックしてアイテムをリストに追加します。

## アイテムを起動する

リスト内のアイテムをクリックすると起動します。
