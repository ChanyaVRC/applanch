# 設定リファレンス

設定はメインウィンドウ右上の設定ボタンから開きます。

---

## 外観

### テーマ

アプリケーションのカラーテーマを変更します。

| 選択肢 | 動作 |
|--------|------|
| システムに従う（初期値） | Windows のライト/ダーク設定に合わせて切り替える |
| ライト | 常にライトパレットを使用する |
| ダーク | 常にダークパレットを使用する |

`Config/UserDefined/theme-palette/*.json` のカスタムテーマもここに表示されます。
詳細は [テーマ](themes.ja.md) を参照してください。

### 言語

| 選択肢 | 動作 |
|--------|------|
| システムに従う（初期値） | Windows の表示言語を使用する |
| English | 常に英語で表示する |
| 日本語 | 常に日本語で表示する |

---

## 起動動作

### 起動後の動作

アイテムを起動したあとに applanch がどう振る舞うかを設定します。

| 選択肢 | 動作 |
|--------|------|
| アプリを閉じる（初期値） | アイテム起動後にウィンドウを閉じる |
| ウィンドウを最小化 | ウィンドウをタスクバーに最小化する |
| 開いたままにする | ウィンドウをそのまま表示し続ける |

### 起動前に確認する

有効にすると、アイテムを起動するたびに確認ダイアログが表示されます。

### 管理者として実行

有効にすると、すべてのアイテムが管理者権限で起動されます。

### 削除前に確認する

有効にすると、アイテムをリストから削除する前に確認ダイアログが表示されます。

## 起動・アップデート

### 起動時にアップデートを確認する

有効（初期値）にすると、applanch の起動ごとに GitHub Releases で新しいバージョンを確認します。

### アップデートのインストール方法

アップデートが検出されたときの動作を設定します。

| 選択肢 | 動作 |
|--------|------|
| 通知のみ | バナーを表示するだけ |
| 手動（初期値） | バナーからユーザーが手動でアップデートを実行する |
| 自動適用 | 自動的にアップデートを適用する |

### デバッグ更新モード

実際のバージョン差分がなくてもアップデート機能をテストできます。
開発用途向けです。

### 起動時に最小化する

有効にすると、メインウィンドウが最小化された状態で起動します。

### Windows 起動時に自動起動する

有効にすると、Windows の起動時に applanch が自動的に起動するよう登録されます。

### 起動時にエクスプローラーのコンテキストメニューへ追加する

有効（初期値）にすると、起動時に「Applanch に登録」をエクスプローラーのコンテキストメニューへ追加します。

### Applanch のコンテキストメニュー項目を削除する

このボタンをクリックすると、既存の Applanch コンテキストメニュー項目をすぐに削除します。

## アイテム表示

### カテゴリーの並び順

| 選択肢 | 動作 |
|--------|------|
| アルファベット順（初期値） | カテゴリーを A〜Z の順で表示 |
| 追加順 | カテゴリーを最初に使用した順で表示 |

### アプリリストの並び順

| 選択肢 | 動作 |
|--------|------|
| 手動（初期値） | ドラッグ＆ドロップで変更した順 |
| 名前順 | 表示名のアルファベット順 |
| カテゴリー→名前順 | カテゴリー順に並べ、同カテゴリー内は名前順 |

### クイック追加サジェスト上限

クイック追加入力欄に入力したときに表示するサジェスト件数を設定します。

| 選択肢 | 動作 |
|--------|------|
| 10 | 少なめの候補を表示 |
| 20 | バランスのよい件数 |
| 30 | より多くの候補を表示 |
| 50（初期値） | 一般的な利用に推奨 |
| 100 | 設定画面で選べる最大件数 |

---

## アイコン・ネットワーク

### HTTP アイコンを取得する

有効（初期値）にすると、アイテムに関連付けられた Web URL からアイコンをダウンロードします。

### プライベートネットワークへの HTTP アイコンリクエストを許可する

有効にすると、プライベート IP 帯（例：192.168.x.x）からのアイコン取得を許可します。
初期値は無効です。

---

## ユーティリティ

| 操作 | 内容 |
|------|------|
| デフォルトにリセット | すべての設定を初期値に戻す |
| 診断情報をコピー | アプリバージョン・OS・.NET バージョン・ロケール・ログフォルダーパス・更新設定を含む診断サマリーをクリップボードにコピーする。バグ報告時に便利。 |
| ログフォルダーを開く | applanch がログファイルを保存するフォルダーをエクスプローラーで開く。 |
| バグを報告 | 問題の詳細を記入できる GitHub Issue テンプレートをブラウザで開く。 |
