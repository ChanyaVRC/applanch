# インストールガイド

## 配布形式

applanch は 2 種類の形式で配布されています。

- ポータブル ZIP パッケージ
- インストーラー EXE パッケージ

## ポータブル ZIP

1. GitHub Releases から ZIP アセットをダウンロードします。
2. 任意のフォルダーに展開します。
3. `applanch.exe` を実行します。

## インストーラー EXE

1. GitHub Releases からインストーラーアセットをダウンロードします。
2. インストーラーを起動します。
3. 現在のユーザーのみ、またはすべてのユーザー向けにインストールするかを選択します。
4. セットアップウィザードの手順に従います。

## データの保存場所

インストール方法に関わらず、ユーザーデータは `%LOCALAPPDATA%\applanch\` に保存されます。

| ファイル | 内容 |
|--------|------|
| `settings.json` | アプリケーション設定 |
| `launch-items.json` | 保存されたアイテムリスト |
| `app.log` | アプリケーションログ |

設定ファイル（テーマ、起動フォールバックルール）は `applanch.exe` と同じ場所の `Config\` フォルダーに保存されます。

## アンインストール

**ポータブル ZIP** — 展開したフォルダーを削除します。
`%LOCALAPPDATA%\applanch\` 配下のユーザーデータは自動的に削除されません。
完全に削除したい場合は、そのフォルダーも手動で削除してください。

**インストーラー EXE** — Windows の **設定** → **アプリ** から **applanch** を選択し、**アンインストール** をクリックします。
`%LOCALAPPDATA%\applanch\` 配下のユーザーデータはアンインストーラーでは削除されません。
必要に応じてそのフォルダーも手動で削除してください。
