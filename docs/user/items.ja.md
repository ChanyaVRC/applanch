# アイテムとカテゴリー

## アイテムを追加する

メインウィンドウ上部のクイック追加入力欄にパスやアプリ名を入力します。
インストール済みアプリや入力履歴をもとに最大 10 件のサジェストが表示されます。
**Enter** を押すか **追加** ボタンをクリックするか、サジェストを選択してアイテムを追加します。

メインウィンドウにファイルまたはフォルダーをドラッグ＆ドロップして追加することもできます。

## アイテムを起動する

アイテムをクリックすると起動します。起動後の動作（終了・最小化・維持）は [設定](settings.ja.md) で変更できます。

## コンテキストメニュー操作

アイテムを右クリック（またはホバー時に表示されるボタンを使用）すると以下の操作が行えます。

| 操作 | 内容 |
|------|------|
| 名前を変更 | リストに表示される名前を変更する |
| カテゴリーを変更 | カテゴリーを割り当てまたは変更する |
| 引数を編集 | 起動時に渡すコマンドライン引数を設定する |
| 管理者として実行 | このアイテムを管理者権限で起動する（1 回限り） |
| 場所を開く | アイテムの親フォルダーをエクスプローラーで開く |
| 削除 | リストからアイテムを削除する |

## 引数

**引数を編集** を使って、起動時にアイテムへ引数を渡せます。

例:

| アイテム | 引数 | 効果 |
|------|------|------|
| Chrome | `--incognito` | シクレットモードで起動 |
| メモ帳 | `C:\logs\app.log` | 特定のファイルを開く |
| PowerShell | `-NoProfile -Command "Get-Date"` | コマンドを直接実行 |
| カスタムツール | `--env production --verbose` | ツールのフラグを指定 |

引数はアイテム起動時にパスの後据にそのまま附加されます。

## カテゴリー

カテゴリーはメインウィンドウの左パネルに表示されます。
カテゴリーをクリックするとアイテムリストを絞り込めます。
**すべて** をクリックすると全アイテムが表示されます。

新規アイテムのデフォルトカテゴリーは **General** です。

## アイテムの並び替え

アイテムをドラッグして並び順を変更できます。
変更は自動的に保存されます。

## アイテムの削除

コンテキストメニューまたはホバー時に表示される削除ボタンで削除できます。

削除後、ウィンドウ下部に通知が表示されます。
通知の **元に戻す** をクリックすると、アイテムを元の場所に即座に復元できます。

設定で [削除前に確認](settings.ja.md) が有効な場合、削除前に確認ダイアログが表示されます。

## 並び順モード

並び順の動作は [設定](settings.ja.md) で変更できます。

**カテゴリーの並び順**

| モード | 動作 |
|--------|------|
| アルファベット順（初期値） | カテゴリーを A〜Z の順で表示 |
| 追加順 | カテゴリーを最初に使用した順で表示 |

**アプリリストの並び順**

| モード | 動作 |
|--------|------|
| 手動（初期値） | ドラッグ＆ドロップで変更した順 |
| 名前順 | 表示名のアルファベット順 |
| カテゴリー→名前順 | カテゴリー順に並べ、同カテゴリー内は名前順 |
