# Windows 統合

applanch は Windows と 2 つの方法で統合されます。エクスプローラーの右クリックメニューへのエントリ追加と、Windows 起動時に自動起動するオプションのスタートアップ登録です。

---

## コンテキストメニュー

applanch を初回起動すると、Windows エクスプローラーのコンテキストメニューに **"Register to applanch"** という項目が自動的に登録されます。
ファイル・フォルダー・実行ファイルを右クリックするとこのエントリが表示され、クリックするとそのアイテムが applanch のリストに追加されます。

### 登録対象

| 対象 | コンテキスト |
|------|------------|
| ファイル（任意の拡張子） | ファイルを右クリック |
| 実行ファイル（`.exe`） | プログラムを右クリック |
| フォルダー | フォルダーを右クリック |
| ディレクトリの背景 | フォルダー内の空白部分を右クリック |

### Windows 11 簡易コンテキストメニュー

Windows 11 では、エクスプローラーが **パッケージ ID** を持つアプリのエントリのみを統合コンテキストメニューに表示します。
applanch は初回起動時にスパース MSIX パッケージを登録してパッケージ ID を取得し、このメニューに表示されるようにします。

!!! note
    Windows 11 の簡易コンテキストメニューへの表示には、スパースパッケージの登録が必要です。
    書き込みアクセスのない環境でポータブルとして起動した場合、エントリはクラシックコンテキストメニューにフォールバックし、**その他のオプションを表示** から引き続きアクセスできます。

### コンテキストメニューの再登録

`applanch.exe` を移動した後などにコンテキストメニューが表示されなくなった場合は、以下のコマンドで再登録できます。

```
applanch.exe --register
```

### コンテキストメニューの削除

コンテキストメニューのエントリを削除するには、以下を実行します。

```
applanch.exe --unregister-context-menu
```

## Windows スタートアップ登録

有効にすると、Windows にサインインするたびに applanch が自動的に起動します。

**有効にする方法:**
**設定** → **スタートアップ** → **Windows 起動時に起動** をオンにします。

この設定は以下のレジストリキーに書き込みます。

```
HKCU\Software\Microsoft\Windows\CurrentVersion\Run
```

設定をオフにするとエントリは削除されます。
インストーラーや管理者権限は不要で、ユーザーごとのスタートアップ登録です。
